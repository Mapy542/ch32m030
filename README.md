# RISC - V FOC BLDC & Stepper Driver

The latest version is the ch32v307drv project. I moved away from the original ch32m030 with integrated gate drivers because of the limited current and voltage capabilities. The ch32v307 has a much more MCU with embedded Ethernet phy. I'm hoping to use Ethernet to build a control system for my laser cutter with a homebrew motion controller board.

I am using the LMG2100 100V 53A GAN Totem pole packages from TI because they do all of the hard parts :P

The idea with all IO is to terminate into shielded rj-45 and use shielded cable. For the safety port, the opto-couplers exist to isolate ground loops with other boards in the cabinet, not user protection; hence the lack of creepage and clearance (<=100V). (Also, I needed area for the traces)


You can grab the datasheet if your curious, but I built a pinmux tool to make pin selection easier, and it gives a good overview of capabilities of the chip. It's the part of CubeMX I enjoy using... but with pin remap option automatic solving
